# Remember Command

Save this to memory: $ARGUMENTS

**Memory types:**
- `fact` - Technical facts (default)
- `decision` - Implementation decisions
- `learning` - Lessons learned
- `preference` - User preferences
- `todo` - Tasks to complete
- `context` - Session context

Saving as: fact
Call `rlm_remember(type="fact", content="$ARGUMENTS")`
